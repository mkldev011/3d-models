## Importing to Roblox

1. Download the .glb files
2. Import to Blender - Export as .fbx
3. Import .fbx to Roblox Studio

## Specs
- Format: GLB (binary glTF)
- Poly count: 200-1200 triangles per model
- Origin: Centered at (0,0,0)
